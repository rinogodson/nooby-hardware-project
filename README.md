# My First Robot Car!! 🤖🚗

Hey guys! This is my line following robot project. 

Just a heads up, there is **NO CODE** in this repo. I didn't program it at all. I just bought a kit with a blue circuit board and soldered all the little resistors and the chip thingy onto it. 

It runs on 2 AA batteries and actually follows a black marker line on the floor! (Most of the time lol). 

I burned my finger once with the soldering iron but it works great!
